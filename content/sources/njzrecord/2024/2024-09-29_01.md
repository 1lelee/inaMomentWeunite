# Team Bunnies 公告及抗议信

**[紧急联署及控告人招募] 致现任 ADOR 代表金周永的抗议信**

**金周永现任 ADOR 代表，必须对 HYBE PR 针对 NewJeans 的负面舆论操控、HYBE 社内排挤行为提出正式抗议并要求纠正措施，同时恢复闵熙珍代表的职位，使 ADOR 恢复正常运营。**

## ⚖️ 一、强烈抗议金周永现任 ADOR 代表未对侵犯和妨碍 NewJeans 演艺活动的行为采取必要措施

### 1. 作为 ADOR 代表的法律义务

ADOR 代表负有在第三方侵犯或妨碍 NewJeans 成员演艺活动时，采取必要措施排除该等侵犯或妨碍的义务。

> “ADOR 与 NewJeans 成员之间签订的专属合同第5条第4款规定，若第三方侵犯或妨碍 NewJeans 的演艺活动，ADOR 有义务采取必要措施排除该等侵犯或妨碍。”（[来源：首尔中央地方法院 2024카합20635 决定书](https://m.blog.naver.com/PostView.naver?blogId=lawyer_han&logNo=223468022555&proxyReferer=https:%2F%2Fwww.google.com%2F&trackingCode=external)）

与此相关，当时法院认为，闵熙珍代表就 HYBE 内 A 组合的相似性等问题通过电子邮件向 HYBE 发出书面通知的行为，是“i) 在 A 组合出道前后，大众中亦存在 A 组合的概念、编舞、服装等与 NewJeans 相似的意见；ii) 作为 ADOR 内部董事兼代表理事，闵熙珍代表负有保护 ADOR 核心资产 NewJeans 价值而采取必要措施的善管注意义务或忠实义务；iii) 闵代表向 HYBE 发送指出 A 组合相似性等问题的电子邮件，可视为履行了股东间合同中关于对 ADOR 产生负面影响时立即书面通知 HYBE 的通知义务”，因此闵代表的行为不构成对 ADOR 的背信行为，反而是其作为代表理事应尽的义务。

然而，NewJeans 粉丝不得不担忧，金周永代表在 NewJeans 当前活动面临多种严重侵犯和妨碍的情况下，是否正采取适当措施，或是否存在怠于履职的情形。

### 2. 就 HYBE PR 针对 NewJeans 及 ADOR 的负面舆论操控提出正式抗议及纠正要求

9月26日，媒体公开了一段录音，内容显示 HYBE PR 负责人针对其旗下厂牌所属艺人 NewJeans，贬低其日本出道成绩，并试图进行“NewJeans 被煤气灯效应操控”的负面舆论操控。更令人震惊的是，录音中还披露，当 HYBE PR 负责人发现相关记者和媒体不听从其意时，便试图“拉拢”对方，甚至提议向该媒体产业部长提供高尔夫招待（[来源：金贤正新闻秀采访全文](https://www.cbs.co.kr/board/view/cbs_P000246_interview?no=168993)）。

从常识来看，哪家大型娱乐公司的宣传负责人会贬低自家艺人的成绩，称其“实际上表现不佳”？然而，NewJeans 粉丝在看到 HYBE 发布的官方反驳声明后更加愤怒。HYBE 并未对此进行反省或采取纠正措施，反而威胁要对记者采取强硬法律应对。

NewJeans 粉丝要求现任 ADOR 代表金周永：HYBE PR 亦承担 ADOR 的分摊费用并负责 ADOR 的宣传事务。金周永代表必须公开，其为保护 NewJeans 的价值，就 HYBE PR 并非积极宣传 NewJeans 的活动与成绩、反而进行负面扩散的行为，采取了何种正式措施。若未对 HYBE PR 采取任何措施，则意味着其作为 ADOR 代表理事，完全未履行排除侵犯和妨碍 NewJeans 活动行为的必要措施，属严重怠于履职。

### 3. 就 NewJeans 成员遭受的 HYBE 社内排挤行为要求查明真相及纠正

9月11日，NewJeans 成员亲自透露，在 HYBE 内部遭到其他团队经纪人人格上的无视和排挤。其向金周永代表反映此事后，却以“没有证据、为时已晚”为由未能得到妥善后续处理。NewJeans 粉丝对金周永代表作为 ADOR 代表理事兼内部董事，未能妥善保护所属艺人的行为深表忧虑。我们要求对此进行彻底的真相调查。

## 💼 二、要求金周永代表及李道京副代表——作为 HYBE 任命的 ADOR 内部董事——赔偿因无计划解雇闵熙珍代表而导致 NewJeans 后续活动、粉丝见面会及专辑制作中断造成的损失，并立即恢复 ADOR 正常运营

刑法第356条（业务上的背信罪）规定，处理他人事务者若违反其任务，为自己或第三方谋取财产利益，从而给本人造成损失，可处以10年以下有期徒刑或3000万韩元以下罚金。

金周永代表作为 ADOR 内部董事，负有保护 ADOR 及 NewJeans 利益与价值的善管注意义务和忠实义务。然而，金周永代表并未维护 ADOR 的经营利益和价值，反而为 HYBE（议长方时赫）之利，在无应对方案的情况下解雇闵熙珍代表，并自行就任 ADOR 代表理事。

由此给 ADOR 造成的财产损失预计高达数百亿韩元。NewJeans 与闵熙珍代表原计划的粉丝见面会及后续专辑制作已告中断。参考数据：NewJeans 2024年上半年的销售额为615亿韩元（同比上一年上半年增长38.9%），净利润为110亿韩元。某证券公司推算，若 NewJeans 明年进行世界巡演，将吸引50至80万观众，后年 ADOR 的营业利润将达到900亿至1000亿韩元，进而将 ADOR 的企业估值推高至2万亿韩元（[来源：韩亚证券研究中心 HYBE 企业分析](https://ssl.pstatic.net/imgstock/upload/research/company/1713229681819.pdf)）。

然而，金周永代表通过给 ADOR 造成高达数百亿韩元的损失，并自行就任 ADOR 代表理事，获取了代表理事职位及薪资等财产利益。这违反了金周永代表作为 ADOR 内部董事的善管注意义务和忠实义务，其行为并非为了 ADOR 的经营利益，而是纯粹出于 HYBE 最高管理层的判断及谋取私利，完全可能构成针对 ADOR 的业务上的背信罪。

NewJeans 粉丝并非仅仅担忧 ADOR 的财产损失。正如第一封公开信所述，对于正处于第三年高速成长期的 NewJeans 而言，当前时期至关重要。这是任何从事娱乐业人士的常识。然而，由金周永等 HYBE 任命的 ADOR 内部董事，在无应对方案的情况下解雇了 ADOR 代表理事兼 NewJeans 总制作人闵熙珍代表，直接导致 NewJeans 的国内粉丝见面会被取消、后续专辑制作中断，对此 NewJeans 粉丝无法不感到愤怒。金周永代表和 HYBE 切勿以“闵熙珍代表拒绝业务劳务合同（制作业务）”为由进行宣传，试图推卸责任。剥夺了权利，又如何要求履行义务？

[NewJeans 粉丝已向 ADOR 现任董事会两次发送载明我们要求的公开信，但至今未收到任何答复。](https://docs.google.com/document/d/1cDKPkQmVmwRMXSFeCSUNgp_f2DIoMCcXrI6akLWB9v8/edit?tab=t.0) 我们对方时赫议长和 HYBE 无视并拒绝 NewJeans 成员要求的行为感到愤怒。

最后，我们再次提出要求：为使 NewJeans 能够脱离其活动受到侵犯和妨碍的母公司 HYBE，正常开展活动，请恢复闵熙珍代表的职位并恢复 ADOR 的正常运营。若这次仍无视粉丝意见，不采取任何措施，我们作为 NewJeans 粉丝警告，将可能对金周永代表等人追究刑事责任的控告措施。
